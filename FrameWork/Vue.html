# 主要内容

  • 在Vue中使用jsx实现模板语法之外的复杂逻辑和功能（开始怀疑自己到底是在写react还是vue）。
  • 实现自定义组件render函数，让组件更自由
  • 实现组件之间Dispatch和BroadCast（向上派发和向下广播）事件
  • 实现简易vue-router和vuex

                state.count++;
            }

                document.body.innerHTML =`
